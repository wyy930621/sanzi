SYSTEM INTRADUTION

系统介绍

运营级智能生活平台开发

介绍：

充分发挥电信运营商的基础网络、维护经验、企业信誉、营业网点、拥有大量接入用户的，同时结合我公司在智能生活领域的的技术、运营模式、用户周边服务以及用户刚性需求优势。

目的：

共同打造一个可以满足不同层次用户需求、价格亲民、使用便捷、维护方便、稳定可靠、具有无限可扩展性的综合服务平台。

前景：

平台运营后，可以为合作电信运营商提供一个全新的、市场空白的、高收益率的、可不断融合新技术、新服务业务的发展平台及发展模式。从而摆脱在互联网接入市场上的毫无休止的价格战，摆脱被网络内容经营商管道化、边缘化的现状，实现电信运营商在物联网大潮下的二次飞跃，最终成为智能生活乃至物联网发展的主体领导者。

SYSTEM ARCHITECTURE

系统架构

NETWORK INTRODUCTION

网络架构

核心层：中心服务器、访问服务器

接入层：接入服务器、Internet

用户层：家居网关、wifi路由器、移动与电脑端、家具传感器、摄像头

核心层、接入层功能 >>>

1.实现大用户量数据的采集、传输、处理、反馈、备份、告警业务信息的发送等。将普通智能家居产品的高成本部分在远端服务器上全部实现，从而为最终用户降低成本。

2.建立完善的运营商业务受理、安装维护、客服、计费、工单流转系统、系统管理账号分级系统、销售资费自定义系统、设备ID分发系统等服务，方便运营商运营、维护。

3.基于用户信息的收集、整理，可以为运营商的下一步商业活动提供细致、全面、具体的定制用户数据报表。

4.建立了贴近用户实际生活的商业服务平台，随着运营商的接入，可以为覆盖用户提供全面、符合实际生活需要的商业信息。

5.建立了开放的接口，可以为正在建设中的智能农业、家庭汽车、身体信息监测等功能实现无缝对接。到时本平台就可以为接入用户提供全方位的从生活到工作、生产的，满足不同生活范围、工作范围人群需要的综合智能生活服务。

用户层功能 >>>

用户层硬件功能：

1.各个传感器模块的信号感知、发送。

2.网关双向通信备份，在家庭常用WIFI接入通道中断的情况下，自动切换到GPRS或4G通信模式，确保用户家庭系统的正常使用。

3.专用的通信协议，专用的数据加密方法确保用户数据的保密性、安全性。

4.设备心跳保活机制，一旦有用户设备模块发生掉电、人为破坏等导致设备无法正常工作的情况，会立即通知用户与设备维护中心。

5.设备电源切换提醒机制，一旦用户设备供电由市电供电切换为电池供电则立即通知用户与维护人员，在电池耗尽前立即处理，以保证设备正常运行，用户的家庭财产人身安全获得持续的保障。

6.建立了完善的设备状态查询功能，维护人员与用户可以在远程对家庭所有的设备的运行状态有清楚的了解，从而将可能发生的故障杜绝，减少用户损失。

7.自动路由选择机制，用户家庭网关掉电或损坏情况下，用户家庭模块会自动选择周围可以检测到的其他家庭网关做临时网关，保证用户使用，直到用户家庭网关恢复使用。

8.独创的用户出入行为判断系统，可以对用户的出入行为进行判断，防止小动物、行人路过等造成的误报警情况的发生。

用户层软件APP功能：

1.家庭系统的撤防、布防。

2.家庭用户的添加、管理。

3.红外家电控制。

4.系统模块状态查询

5.告警信息查询、接受、管理。

6.日志信息的查询、管理。

7.系统状态查询。

8.情景模式设置。

9.周边商业服务信息的查询与接受。

10.通过用户APP,可以建立起用户与系统，用户与运营商的沟通渠道。

11.系统通知信息的接收与管理。
